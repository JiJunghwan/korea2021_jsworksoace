대입연산자란?
변수에 값을 할당하는 방법과 관련된 연산자

=       ex) var x = 5; // 변수에 값을 대입하는 연산자
+=     ex) x+=3;         x = x+3 과 동일표현식의 단축표현
-=      ex) x-=3;           x = x-3 과 동일
*=      ex) x*=3;           x= x*3 과 동일
/=      ex) x/=3;           x=x/3 과 동일
%=      ex) x%=3;       x=x%3; 과 동일 

ex ) x =5   [ x%=7] = 5%7
